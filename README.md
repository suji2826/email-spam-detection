# Email Spam Detection

## Overview
This project uses Machine Learning to classify emails as Spam or Ham.

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## Features
- Email classification
- Text preprocessing
- Accuracy evaluation

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run

python spam_detection.py
